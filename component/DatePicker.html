DatePicker 日期选择器 ​

用于日期选择，与官方日期选择的区别是：只支持长整型的时间戳值,支持字符串类型和数字类型的时间戳

其它配置参数参考官方日期选择器

基础用法 ​


选中的日期时间戳：
vue
<script lang="ts" setup>
import { ref } from 'vue'
const dateValue = ref<string>()
</script>
<template>
  <hg-date-picker v-model="dateValue" />
  <br />
  选中的日期时间戳：{{ dateValue }}
</template>
