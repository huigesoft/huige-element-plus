IconPicker Base64图片选择器 ​

用于选择图片，可直接生成Base64编码的图片字符串，常用于小图片存储到数据库等。

用法 ​

启用预览


图片Base64编码值：
vue
<script lang="ts" setup>
import { ref } from 'vue'
const imgData = ref<string>()
const preview = ref(true)
</script>
<template>
  <el-switch v-model="preview" inline-prompt active-text="启用预览" inactive-text="禁用预览" />
  <br />
  <hg-base64-img v-model="imgData" :preview="preview"></hg-base64-img>
  <br />
  图片Base64编码值：<el-input type="textarea" v-model="imgData" rows="4" />
</template>
