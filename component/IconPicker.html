IconPicker 图标选择器 ​

用于选择或输入图标

基础用法 ​


选中的图标值：
选中的图标是：
vue
<script lang="ts" setup>
import { ref } from 'vue'
const iconValue = ref<string>()
</script>
<template>
  <hg-icon-picker clearable style="width: 200px" v-model="iconValue"></hg-icon-picker>
  <br />
  选中的图标值：{{ iconValue }}
  <br />
  选中的图标是：<hg-icon :type="iconValue" />
</template>

自定义分页 ​

通过 page-size 属性配置下拉框中每页图标数量

vue
<script lang="ts" setup>
import { ref } from 'vue'
const iconValue = ref<string>()
</script>
<template>
  <hg-icon-picker
    clearable
    style="width: 200px"
    v-model="iconValue"
    :page-size="33"
  ></hg-icon-picker>
</template>

自定义下拉框宽度 ​

通过popper-width属性定义下拉框宽度

vue
<script lang="ts" setup>
import { ref } from 'vue'
const iconValue = ref<string>()
</script>
<template>
  <hg-icon-picker
    clearable
    style="width: 200px"
    v-model="iconValue"
    :popper-width="300"
    :page-size="30"
  ></hg-icon-picker>
</template>
