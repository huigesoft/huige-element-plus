Modal 模态窗口 ​

Modal 模态窗口，兼容ElementPlus 组件库的 Dialog 和 Drawer 两个组件

基础用法 ​

vue
<script lang="ts" setup>
import { ref } from 'vue'
const visible = ref(false)
const type = ref(0)
const width = '50%'

const showDialog = () => {
  type.value = 1
  visible.value = true
}

const showDrawer = () => {
  type.value = 0
  visible.value = true
}
</script>
<template>
  <hg-modal v-model="visible" :type="type" :width="width"> 我是框里内容 </hg-modal>
  <el-button @click="showDialog">打开Dialog</el-button>
  <el-button @click="showDrawer">打开Drawer</el-button>
</template>

API ​

Attributes ​

属性名说明类型默认值
model-value / v-model绑定值，是否显示booleanfalse
type弹窗类型，0：Drawer，1：Dialognumber0
width宽度,字符串需设置成%，数字为像素number | string50%

其它属性参考 ElementPlus 组件库中的 Dialog 和 Drawer
