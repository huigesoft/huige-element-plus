Splitter 分割面板 ​

可将区域水平或垂直分隔，并可自由拖动以调整各个区域的大小。

基础用法 ​

最基本的用法

Left Panel
Right Panel
vue
<script lang="ts" setup></script>
<template>
  <hg-splitter
    style="width:100%;height:300px;box-shadow: var(--el-border-color-light) 0px 0px 10px;"
  >
    <hg-splitter-left> Left Panel </hg-splitter-left>
    <hg-splitter-right>Right Panel</hg-splitter-right>
  </hg-splitter>
</template>

垂直布局 ​

通过配置layout来控制是否垂直。layout=v 垂直分割，layout=h 水平分割

水平分割子组件只能是 hg-splitter-left 和 hg-splitter-right

垂直分割子组件只能是 hg-splitter-top 和 hg-splitter-bottom

Top Panel
Bottom Panel
vue
<script lang="ts" setup></script>
<template>
  <hg-splitter
    :split-size="150"
    layout="v"
    style="width:100%;height:300px;box-shadow: var(--el-border-color-light) 0px 0px 10px;"
  >
    <hg-splitter-top> Top Panel </hg-splitter-top>
    <hg-splitter-bottom> Bottom Panel</hg-splitter-bottom>
  </hg-splitter>
</template>

可折叠 ​

配置 handler-show 可提供快速收缩功能。

启用折叠
Left Panel
Right Panel
vue
<script lang="ts" setup>
import { ref } from 'vue'
const handlerShow = ref<boolean>(true)
</script>
<template>
  <el-switch v-model="handlerShow" inline-prompt active-text="启用折叠" inactive-text="禁用折叠" />
  <hg-splitter
    :handler-show="handlerShow"
    style="width:100%;height:300px;box-shadow: var(--el-border-color-light) 0px 0px 10px;"
  >
    <hg-splitter-left> Left Panel </hg-splitter-left>
    <hg-splitter-right>Right Panel</hg-splitter-right>
  </hg-splitter>
</template>

可拖动 ​

配置 resizable 属性可以控制是否允许拖动。

启用拖拽
Left Panel
Right Panel
vue
<script lang="ts" setup>
import { ref } from 'vue'
const resizable = ref<boolean>(true)
</script>
<template>
  <el-switch v-model="resizable" inline-prompt active-text="启用折叠" inactive-text="禁用折叠" />
  <hg-splitter
    :resizable="resizable"
    style="width:100%;height:300px;box-shadow: var(--el-border-color-light) 0px 0px 10px;"
  >
    <hg-splitter-left> Left Panel </hg-splitter-left>
    <hg-splitter-right>Right Panel</hg-splitter-right>
  </hg-splitter>
</template>

插槽 ​

可以配置收缩和展开图标插槽，仅在type为left和top的 splitter-panel上起作用。

Left
Top Panel
Bottom Panel
vue
<script lang="ts" setup>
import { ref } from 'vue'
const resizable = ref<boolean>(true)
</script>
<template>
  <hg-splitter
    style="width:100%;height:300px;box-shadow: var(--el-border-color-light) 0px 0px 10px;"
  >
    <hg-splitter-left>
      <template #expanded-icon>
        <hg-icon type="caret-left" />
      </template>
      <template #collapsed-icon>
        <hg-icon type="caret-right" />
      </template>
      Left
    </hg-splitter-left>
    <hg-splitter-right>
      <hg-splitter
        layout="v"
        style="width:100%;height:300px;box-shadow: var(--el-border-color-light) 0px 0px 10px;"
      >
        <hg-splitter-top>
          <template #expanded-icon>
            <hg-icon type="caret-left" />
          </template>
          <template #collapsed-icon>
            <hg-icon type="caret-right" />
          </template>
          Top Panel
        </hg-splitter-top>
        <hg-splitter-bottom>Bottom Panel</hg-splitter-bottom>
      </hg-splitter>
    </hg-splitter-right>
  </hg-splitter>
</template>

Splitter API ​

Attributes ​

属性名说明类型默认值
v-model:collapsed收缩状态,true:收缩，false:展开booleanfalse
v-model:split-size分割大小，第一个面板的宽度/高度（px）number250
collapsed-size收缩后，第一个面板收缩的大小（px）number0
handler-size拖动条的宽度 （px）number3
handler-show是否显示快速收缩图标,显示：可以快速收缩，不显示：不允许收缩booleantrue
resizable是否允许拖拽调整大小booleantrue
layout分割方向，h:水平分割，v:垂直分割enum('h' | 'v')true

Splitter Panel API ​

Splitter Panel 共包含个组件，左右分割：hg-splitter-left 和 hg-splitter-left ；上下分割：hg-splitter-top 和 hg-splitter-bottom

Splitter Panel Slots ​

插槽只存在于类型为“left” 和 “top” 的面板上，提供自定义点击图标入口

插槽名说明
expanded-icon展开后的插槽，可以自定义图标
collapsed-icon收缩后的插槽，可以自定义图标

Splitter Panel Event ​

事件只存在于类型为“left” 和 “top” 的面板上

事件名说明类型
resize-start开始拖拽调整大小的回调函数Function ，(size: number) => void
resize拖拽调整大小过程中的回调函数Function ，(size: number) => void
resize-end结束拖拽调整大小的回调函数Function ，(size: number) => void
collapse收缩/展开 状态发生改变的回调函数Function ，(collapsed: boolean) => void
