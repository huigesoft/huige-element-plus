快速开始 ​

本节将介绍如何在项目中使用 huige-element-plus UI 扩展组件库。

快速开始 ​

该组件库是ElementPlus的扩展增强组件库，使用前，必须保证已正确引入并使用ElementPlus组件库。

javascript
import 'element-plus/dist/index.css'
import 'huige-element-plus/dist/index.css'
import ElementPlus from 'element-plus'
import HuigeElementPlus from 'huige-element-plus'

const app = createApp(App)
app.use(ElementPlus)
app.use(HuigeElementPlus)
app.mount('#app')

示例 ​

代码(图标选择器) ​

vue
<script lang="ts" setup>
import { ref } from 'vue'
const iconValue = ref<string>()
</script>
<template>
  <hg-icon-picker clearable style="width: 200px"></hg-icon-picker>

  <br />
  <hg-icon-picker clearable style="width: 200px" v-model="iconValue"></hg-icon-picker>

  选中的图标值为：{{ iconValue }}
  <br />
  选中的图标展示位：
  <hg-icon :type="iconValue" />
</template>
